# Project Abstract: CyberShield AI

**Title:** CyberShield AI – Real-Time DDoS Threat Intelligence & Security Operations Center (SOC)

**Abstract:**
With the exponential increase in global cyber threats, traditional monitoring tools often lack the real-time responsiveness and intuitive visualization required by modern Security Operations Centers (SOC). This project, **CyberShield AI**, presents a comprehensive, full-stack enterprise web application designed to simulate, monitor, and analyze Distributed Denial of Service (DDoS) attacks in real-time. 

Developed utilizing a modern technology stack—React, Three.js, FastAPI, and PostgreSQL—the system features an interactive 3D WebGL globe that maps incoming attack vectors across international borders. The backend architecture relies on asynchronous Python processing and WebSockets to push live data streams instantly to the client, minimizing latency. Furthermore, the application integrates heuristic AI analytics to compute dynamic Threat Levels and System Health scores based on historical packet rates and attack severities. 

The resulting platform serves as both a high-performance demonstration of full-stack engineering and an educational tool for visualizing complex network topologies and threat intelligence data within a secure, authenticated environment.
